# 📊 HR Analytics Dashboard (Power BI)

An interactive HR Analytics Dashboard built using **Microsoft Power BI** to analyze employee attrition, workforce demographics, salary distribution, and job roles. This dashboard provides HR professionals and management with valuable insights for workforce planning and decision-making.

---

## 📌 Dashboard Preview

![HR Analytics Dashboard](dashboard.png)

> Replace `dashboard.png` with your dashboard screenshot filename if necessary.

---

## 🚀 Features

- Employee Count Overview
- Total Attrition Analysis
- Average Monthly Income
- Average Employee Age
- Attrition by Gender
- Attrition by Age Group
- Attrition by Salary Slab
- Attrition by Job Role
- Attrition by Years at Company
- Education Field Filter
- Interactive slicers and visualizations

---

## 📈 Dashboard Metrics

| Metric | Description |
|---------|-------------|
| Employee Count | Total number of employees |
| Attrition | Total employees who left the organization |
| Average Income | Average monthly salary |
| Average Age | Average employee age |
| Years at Company | Average years employees worked |

## 📚 Insights

The dashboard helps answer questions such as:

- Which job roles have the highest attrition?
- Which salary range experiences the most employee turnover?
- Which age group has the highest attrition?
- How does attrition vary by gender?
- Which education fields contribute most to attrition?
- How does employee tenure affect attrition?
